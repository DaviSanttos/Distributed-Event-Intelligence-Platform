# Distributed-Event-Intelligence-Platform

## Visão Geral
Sistema para gerenciamento de eventos educacionais, com foco em
check-in, controle de presença e geração de certificados.

## Objetivo
Servir como projeto prático para estudo de arquitetura,
clean code e boas práticas de engenharia de software.

## Escopo Inicial
- Cadastro de eventos
- Cadastro de participantes
- Check-in
- Relatórios básicos

## Status
Em fase inicial de fundação.
